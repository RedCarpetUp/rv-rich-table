## How to run

`yarn install`
`yarn start`

## Description

This project uses the github API.

## Features

1. resizable columns
2. editable fields
3. collapsible rows
4. copy paste from excel to table
5. all data from API.
6. Spinner when loading data
7. pagination
8. buttons in cells with modals and sidebars
